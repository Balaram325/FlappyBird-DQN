# 🐦 Flappy Bird AI using Deep Q-Network (DQN)

This project implements a Reinforcement Learning agent that learns to play Flappy Bird using Deep Q-Learning.

## 🚀 Features
- Deep Q-Network (DQN)
- Experience Replay
- Epsilon-Greedy Strategy
- Trained using Google Colab GPU

## 🧠 How it works
The agent interacts with the environment and learns optimal actions over time by maximizing cumulative reward.

## 🛠 Tech Stack
- Python
- PyTorch
- Gymnasium
- Google Colab
